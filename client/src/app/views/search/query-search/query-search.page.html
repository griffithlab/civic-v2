CIViC Advanced Search

Use Advanced Search to locate CIViC entities and artifacts by specifying filters for the attributes relevant to your interest.

Instructions

Select the entity or artifact you wish to search using the tabs. Note that on narrower displays, some tabs may be hidden. In this case, you may select those tabs by scrolling the tabs horizontally, or by clicking the ... overflow button and selecting from the dropdown menu.

Click the 'Add Filter' button to create an initial attribute filter. Then use the 'Select Attribute' selector to choose an attribute you wish to specify for your search. You may specify as many filters as you wish but keep in mind that the narrower your search, the less likely it will return results.

The Match toggle in the header switches between all and any modes. With all selected, the search will only return entities where all specified filters match. With any selected, searches will return entities if any of the specified filters match.

After you've specified your search, click the 'Search' button to query the CIViC knowledgebase. Search results will appear in a table. On wide screens, the table will appear adjacent to the query builder. On narrower screens, the results table will appear beneath.
